🚀 자동 더치페이 앱 (Auto DutchPay App)

결제 즉시, 자동으로 더치페이가 이루어지는 혁신적인 모바일 서비스

팀 무한루프
김남형 · 송지석 · 모준호 · 정권호 · 조영준

📖 프로젝트 소개

친구·동료와 더치페이를 할 때
카카오페이, 토스 등으로 직접 송금하고 정산해야 하는 불편함이 존재합니다.

“얼마 보내면 돼?”
“나중에 줄게…”
→ 이런 스트레스 요소들을 완전히 제거한 자동 정산 앱을 만들었습니다.

결제가 발생하면 앱이 이벤트를 감지하고,
자동으로 금액을 계산해 참여 멤버의 계좌에서 즉시 송금되도록 설계했습니다.
(금융 API가 필요한 영역은 시뮬레이션 기반 처리)

🧪 구현 범위 (Real vs Simulation)
기능	구현 여부	설명
UI / 화면 구성	✔️	Figma 기반 UI + 실제 앱 형태 구현
자동 더치 계산 로직	✔️	인원별/비율별 계산 로직 완성
NFC 참여 플로우	✔️	NFC 이벤트 UI & Simulated Flow
결제 이벤트 감지	🔄(시뮬)	금융 API 필요 → Mock 처리
실제 자동 송금	🔄(시뮬)	오픈뱅킹 API 필요로 인해 시연용 시뮬레이션
백엔드 서버	✔️	정산 로직 구조 및 상태 관리 구현
💡 기술 스택 (Tech Stack)

Frontend: React Native / Expo

Backend: Node.js (정산 로직 설계 / API Mock)

Auth: 자체 본인 인증 플로우 구현

Infra: Local Simulation 기반

기타: NFC/지문 인증 시나리오 처리

🧭 서비스 차별점 (Why This App?)

PDF + 발표 대본 기반으로 정리.

기존 서비스(토스·카카오페이)는
“사용자가 직접 송금해야 하는 구조”

이 앱은
“결제 → 자동 참여 → 자동 정산 → 알림”
완전 자동 구조를 갖추고 있음

결제 후 10초 안에 정산이 끝나는 초고속 자동화

대학생/직장인 모두에게 실사용 가치 높음
